# TodoList 프로젝트

**간단한 TodoList 웹 애플리케이션**

이 프로젝트는 사용자가 할 일을 추가, 수정, 삭제할 수 있는 기능을 제공하는 미니 프로젝트입니다. Go와 React를 사용하여 개발되었으며, MySQL을 데이터베이스로 사용합니다.

## 주요 기능

- 할 일 추가
- 할 일 목록 보기
- 할 일 수정
- 할 일 삭제
